# Milestone 1 & 2 Self-Test Checklist

## Routes Mount Without Errors (Milestone 1 & 2)
- [ ] App starts without console errors
- [ ] All routes load successfully: /, /chat, /today, /journal, /gratitude, /progress
- [ ] 404 page loads for invalid routes
- [ ] Root route (/) redirects or shows Chat page
- [ ] /chat route displays Chat page

## Header is Visible (Milestone 1)
- [ ] Header displays "Balance Agent" title
- [ ] Header is sticky at top of page
- [ ] Logo/icon is visible

## Nav is Clickable (Milestone 1 & 2)
- [ ] All navigation items (Chat, Today, Journal, Gratitude, Progress) are clickable
- [ ] Navigation items have 44px minimum height for mobile tap targets
- [ ] Clicking nav items navigates to correct pages
- [ ] Mobile navigation scrolls horizontally without visible scrollbar
- [ ] All nav tabs switch between pages correctly

## Active Route Highlighting (Milestone 1 & 2)
- [ ] Active route is highlighted (border and text color change)
- [ ] Active state persists when navigating between routes
- [ ] Correct route is highlighted on page load
- [ ] Active state is visible on mobile screens

## Chat Page (Milestone 2)
- [ ] Chat.tsx exists at src/pages/Chat.tsx
- [ ] Chat.tsx composes AvatarChat component
- [ ] Chat page renders without errors
- [ ] All chat functionality works (send messages, receive responses, etc.)

## Additional Checks
- [ ] No TypeScript errors
- [ ] No ESLint errors
- [ ] Layout wrapper applies consistently to all routes
- [ ] Mobile-first spacing is applied
- [ ] No console errors when navigating
- [ ] All routes render their respective components
